---
title: "Get Article by ID"
api: "GET https://api.tradingnews.press/v1/news/{id}"
description: "Retrieve a single article by its ULID"
---

## Path Parameters

<ParamField path="id" type="string" required>
  The article's ULID. Example: `01KNKV33C9DDADR1HRRWGCSA8V`
</ParamField>

## Response

Returns the full article object, or `404` if not found.

<RequestExample>

```bash cURL
curl -H "X-API-Key: YOUR_KEY" \
  "https://api.tradingnews.press/v1/news/01KNKV33C9DDADR1HRRWGCSA8V"
```

```python Python
import httpx

resp = httpx.get(
    "https://api.tradingnews.press/v1/news/01KNKV33C9DDADR1HRRWGCSA8V",
    headers={"X-API-Key": "YOUR_KEY"}
)
article = resp.json()
```

</RequestExample>

<ResponseExample>

```json 200
{
  "id": "01KNKV33C9DDADR1HRRWGCSA8V",
  "content": "BREAKING: Fed announces emergency rate cut of 50bps",
  "urgency": "breaking",
  "sentiment": null,
  "published_at": "2026-04-07T14:30:00+00:00",

```


</ResponseExample>
